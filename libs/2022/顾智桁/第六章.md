# 树和二叉树

## 树

### 查找类

1. **Root(T)** // 求树的根结点
2. **Value(T, cur_e)** // 求当前结点的元素值
3. **Parent(T, cur_e)** // 求当前结点的双亲结点
4. **LeftChild(T, cur_e)** // 求当前结点的最左孩子
5. **RightSibling(T, cur_e)**  // 求当前结点的右兄弟
6. **TreeEmpty(T)**  // 判定树是否为空树
7. **TreeDepth(T)**  // 求树的深度
8. **TraverseTree( T, Visit() )**  // 遍历
